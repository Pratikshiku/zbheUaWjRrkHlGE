## Student Dormitory Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/47e67006d35447f4855465503b213e03/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (any of the listed software can be used)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Functional Description
The system is divided into (roles): regular users (students), dormitory staff, administrators.

**Regular User Functions:**
1. Login
2. Registration
3. Change Password
4. Personal Information
5. Student Information Management
6. Apartment Asset Information
7. Payment Information
8. Public Area Cleaning Information
9. Daily Affairs Information
10. Bed Assignment Information


**Dormitory Staff Functions:**
1. Login
2. Registration
3. Change Password
4. Personal Information
5. Student Management
6. Student Information Management
7. Apartment Asset Management
8. Asset Damage Management
9. New Asset Management
10. Payment Information Management
11. Public Area Cleaning Management
12. Daily Affairs Management
13. Bed Assignment Management


**Administrator Functions:**
1. Login
2. Change Password
3. Student Management
4. Dormitory Staff Management
5. Administrator Management
